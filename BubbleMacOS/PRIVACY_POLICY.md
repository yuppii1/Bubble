# Privacy Policy for AITagger

**Last Updated: February 28, 2026**

AITagger is committed to protecting your privacy. This Privacy Policy explains how our desktop application handles information.

## 1. Information Collection
**AITagger does not collect, store, or transmit any of your personal data to our own servers.**

## 2. Information Processing
To provide its core functionality (analyzing and tagging files), AITagger processes the following:
- **File Content:** Only the content of the files you explicitly select for scanning is read.
- **Provider API Keys:** API keys (Gemini, OpenAI, Anthropic) are stored locally on your device (macOS Keychain/UserDefaults or Linux config file) and are only used to authenticate requests to the respective AI providers.

## 3. Third-Party Services
AITagger communicates with third-party AI providers that you choose to use. These providers have their own privacy policies:
- [Google Privacy Policy](https://policies.google.com/privacy)
- [OpenAI Privacy Policy](https://openai.com/privacy)
- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)

## 4. Security
We store your API keys locally using industry-standard encryption where available (e.g., macOS Keychain). 

## 5. Changes to This Policy
We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

## 6. Contact Us
If you have any questions about this Privacy Policy, please contact the maintainers via the GitHub repository.
