# v0.0.2 - Persistent Summaries 🚀

This release introduces **Persistent Summaries**, which intelligently caches AI-generated metadata locally to save time and tokens.

## 🌟 What's New
- **Persistent Summary Cache:** Automatically stores file summaries in a hidden `.bubble/summaries.json` folder.
- **Incremental Scanning:** Only processes files that have been modified since the last scan (tracks size and modification date).
- **Cross-Platform CLI:** Native support for Linux (AMD64/ARM64), macOS (ARM64), and Windows (AMD64).
- **Improved macOS App:** Enhanced menu bar experience with better metadata handling.

## 📦 Build Artifacts
- **macOS Installer:** `Bubble.dmg`
- **Linux Binaries:** `bubble-linux-amd64`, `bubble-linux-arm64`
- **Windows Binary:** `bubble-windows-amd64.exe`
