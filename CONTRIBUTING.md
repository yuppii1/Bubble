# Contributing to AITagger

First off, thank you for considering contributing to AITagger! It's people like you that make AITagger such a great tool.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include as much detail as possible, including:
- Your OS version.
- The version of AITagger you are using.
- Steps to reproduce the bug.

### Suggesting Enhancements
We love new ideas! If you have a suggestion for an enhancement, please open an issue with the "enhancement" label.

### Improving Documentation
Documentation is crucial for any open-source project. If you find a typo or think a section could be clearer, please submit a pull request!

## Pull Request Process
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. If it's a new feature, add relevant tests.
4. Update the README if necessary.
5. Submit a pull request to the `main` branch.

## Building from Source
To build the Go CLI:
```bash
cd AITaggerLinuxGo
go build -o aitagger main.go
```

To build the macOS app (Swift):
```bash
cd AITagger
swift build
```
